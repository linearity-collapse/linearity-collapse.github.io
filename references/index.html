역사적인 자료들

  • UNIX System V Release 4 Programmer's Guide: ANSI C and Programming Support Tools (ISBN 0-13-933706-7): appendix B에 mapfile이라는 이름으로 link editor 명령이 소개됩니다.
  • UNIX System V Release 4 Programmer's Reference Manual (ISBN 0-13-947029-8): 당시에 쓰였던 ar 파일, SCSS 파일, 그리고 ELF 파일 등의 형식 정보를 정리해둔 장이 있습니다.
  • SCO OpenServer 5 Documentation: Link Editor Command Language: link editor가 어떤 형태로 제공되었는지 엿볼 수 있는 자료입니다.
  • Solaris (Sun OS) Assembler Manual: 어셈블러 syntax를 어떻게 정의했었는지 참고할 수 있습니다.
  • UNIX Assembler Reference Manual: PDP-11 용으로 작성되었던 오리지널 레퍼런스 같습니다. 이유는 모르겠지만 원본은 잘 안 보이고 TeX으로 재조판된 버전만 인터넷에 보입니다.
  • Altos UNIX System V Support Tools Guide: 여기도 link editor와 관련된 장이 있습니다.
  • Digital UNIX Assembly Language Programmer's Guide: AT&T syntax로 매뉴얼이 적혀 있습니다.
  • UNIX Programmer's Manual (1973년 출간): a.out과 같은 파일 형식에 대한 기록이 남아 있습니다.
  • Intel ASM86 Language Reference Manual: Intel이 공식 출간한 레퍼런스 매뉴얼로, Intel syntax로 매뉴얼이 작성되어 있습니다.

기계 학습 관련

  • Neural Networks and Deep Learning: 인공 신경망을 활용한 딥러닝 입문서입니다. 무료 온라인 북입니다.
  • Full-Stack, GPU-based Acceleration of Deep Learning: NVIDIA에서 만든 강의로, 하드웨어를 이용해 딥러닝을 가속하는 방법에 대해 설명합니다.
  • MIT Introduction to Deep Learning: MIT에서 만든 딥러닝 입문 강의입니다. Google Colab에서 바로 실행해볼 수 있는 Jupyter Notebook 파일이 함께 제공됩니다.

GPU

내부 아키텍처와 관련된 글

  • Gentle introduction to GPUs inner workings: GPU 내부의 작동 방식에 대해 (약간의 추측을 더해) 설명하는 글입니다. NVIDIA GPU 뿐만 아니라 AMD GPU, Intel GPU에 대해서도 다룹니다.
  • Exploring the GPU Architecture: VMWare에서 작성한 GPU 내부 아키텍처 설명문입니다. 가볍게 읽기에 좋습니다.
  • GPU Architecture Explained: Everything You Need to Know and How It Has Evolved: NVIDIA GPU 내에서 연산이 어떻게 이루어지는지와 역대 아키텍처에 대해 간단한 설명이 담겨 있는 글입니다.

NVIDIA

  • NVIDIA Technologies: NVIDIA GPU와 관련된 기술 (아키텍처, SDK 등) 목록이 쓸데없는 마케팅 용어 없이 깔끔하게 정리되어 있는 페이지입니다.
  • A History of NVidia Stream Multiprocessor: NVIDIA의 SM (Stream Multiprocessor)이 어떻게 생겨났고, 어떻게 진화해왔는지 잘 정리되어 있는 글입니다.
  • An Introduction to Modern GPU Architecture: 2008년에 작성된 NVIDIA GPU 아키텍처 소개글입니다.
